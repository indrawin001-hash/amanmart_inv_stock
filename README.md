# Stock Opname System

Static web app for retail or supermarket stock opname.

## Open locally

Open `index.html` directly, or serve the folder through a web server:

```powershell
node server.js
```

Then open:

```text
http://localhost:4173
```

## Publish

This folder can be uploaded as a static website to Vercel, Netlify, Cloudflare Pages, GitHub Pages, or any web server. Use `outputs/stock-opname-app` as the site root.
